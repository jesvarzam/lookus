¡Bienvenido a LookUS!


¿Qué ofrecemos?

LookUS es una aplicación web que permite a los usuarios registrados de la empresa LookUS detectar los dispositivos que están en su zona de trabajo.

¿Cómo funciona?

El usuario registrado podrá registrar uno o varios dispositivos en el sistema, los cuales podrán ser detectados posteriormente con el fin
de saber si se trata de un router, una cámara de seguridad o una impresora.

Detecta

Usa nuestro servicio inteligente para detectar los dispositivos conectados a tu red de manera que puedas conocer de primera mano si se tratan de cámaras de seguridad, impresoras o páginas webs personales.

Genera reportes

LookUS generará por ti una serie de reportes con los resultados de cada detección, entre los que se incluyen la fecha de la misma, los puertos abiertos de cada dispositivo o las cabeceras HTTP encontradas.

Entrena

LookUS posee diccionarios preparados para realizar las detecciones. ¿No has quedado satisfecho con alguna detección? Puedes añadir tus propios dispositivos para entrenar nuestro modelo de datos y generará detecciones mucho más precisas.
